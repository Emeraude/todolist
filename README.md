# Todolist

A simple todo list manager.

## Installation

```bash
npm install
```

You can also add it to your PATH or make a symbolic link.

## Synopsis

```
./index.js [-c|--color] [--nocolor] [add <message> [<severity>]] [list [<severity>]] [count [<severity>]] [delete <id>] [drop]...
```
